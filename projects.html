Exam Copy Detection Using AI

This project leverages deep learning to identify potential malpractice in written exam copies. By analyzing scanned handwritten answer sheets, the system detects similarities, repeated content, and abnormal writing styles across different papers. The dataset includes hundreds of annotated exam sheets labeled for similarity. This solution aids universities and boards in ensuring academic integrity at scale.

We trained our model using a dataset of over 1000 scanned exam sheets collected from various institutions. These images were manually annotated to highlight regions of interest such as writing similarities, answer structure, and suspected copied sections. Using convolutional neural networks (CNNs), the model was trained to detect visual patterns, handwriting overlaps, and layout duplications. The training process involved:

  • Preprocessing: Noise removal, binarization, and alignment correction.
  • Augmentation: Rotation, scaling, and brightness variation to improve generalization.
  • Training: Fine-tuning a ResNet-based architecture using supervised learning with cross-entropy loss.

Identifying Cleanliness of Closed Water Tank Using Machine Learning

This system uses image classification and environmental sensors to determine if a closed water tank requires cleaning. By training on images from clean, semi-clean, and dirty tanks, the model predicts cleanliness level with high accuracy. This helps building managers and rural agencies automate water hygiene checks without opening tanks.

Our dataset included more than 1000 labeled images of water tanks in various conditions — clean, semi-clean, and dirty. These images were collected over several months using smartphones and inspection cameras in rural and urban areas. The images were categorized based on expert feedback. The ML model was trained using:

  • Image classification techniques using MobileNet and SVM classifiers.
  • Features like algae patterns, turbidity, and sediment marks were extracted automatically.
  • Data augmentation techniques like mirroring and cropping helped improve model robustness.
